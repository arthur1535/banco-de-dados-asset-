Banco de Dados Asset — Volt

Este site documenta o schema PostgreSQL volt e aponta para o script SQL do repositório.

Ver script SQL

Visão geral

  • Schema: volt
  • Extensão: citext
  • Tabelas: usuarios, clientes, ativos, transacoes, consultas
  • Triggers: atualização de updated_at, normalização de e-mail, sanitização de documento, status default e prevenção de posição negativa

Como aplicar localmente

  1. Crie o banco: CREATE DATABASE volt_capital;
  2. Execute o arquivo SQL no banco volt_capital.

Validações rápidas

SELECT schema_name FROM information_schema.schemata WHERE schema_name = 'volt';
SELECT extname FROM pg_extension WHERE extname = 'citext';
SELECT COUNT(*) FROM information_schema.tables WHERE table_schema = 'volt';
